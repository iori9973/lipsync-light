# LipSync Light

VRChat アバターのリップシンクパラメーター（`Voice` / `Viseme`）に連動して、マテリアルのエミッション（発光色）を制御する Animator 設定を自動生成する Unity Editor 拡張です。

口を動かすと光る目、声に合わせて輝くアクセサリーなど、リップシンク連動の演出を手軽に実現できます。

---

## 動作環境

- Unity 2022.3.22f1
- VRChat SDK（VRCSDK3-AVATAR）3.5.0 以上

---

## インストール

### VCC（VRChat Creator Companion）経由

1. VCC の **Settings → Packages → Add Repository** を開く
2. 以下の URL を入力して「Add」

```
https://iori9973.github.io/lipsync-light/index.json
```

3. プロジェクトの Manage Packages から **LipSync Light** を追加

## 基本的な使い方

### 1. ウィンドウを開く

Unity メニューバーの **Tools → LipSync Light** を選択します。

### 2. Avatar Root を設定する

**Avatar Root** フィールドに、アバターのルートとなる GameObject を Hierarchy からドラッグ&ドロップします。

### 3. モードを選択する

| モード | 内容 |
|---|---|
| **Voice 連動** | `Voice` パラメーター（0〜1）の大きさに応じてエミッション強度をブレンド。声が大きいほど明るく光る |
| **Viseme カラー** | `Viseme` パラメーター（0〜14）の口の形に合わせてエミッションカラーを切り替える |
| **両方** | Voice 連動と Viseme カラーを同時に使用する |

### 4. ターゲットを追加する

**[+ ターゲットを追加]** ボタンで光らせたい Renderer を追加します。

各ターゲットには以下を設定します：

| 項目 | 説明 |
|---|---|
| **Renderer** | 光らせたいメッシュの Renderer コンポーネント |
| **Material** | 対象のマテリアルインデックス（複数マテリアルがある場合にドロップダウンで選択） |
| **Property** | エミッションのシェーダープロパティ名。Renderer を選択すると自動でシェーダーの Color プロパティ一覧がドロップダウン表示される |

> **Property について**
> 多くのシェーダー（Standard / lilToon / Poiyomi）では `_EmissionColor` がエミッション用プロパティです。Renderer を選択するとシェーダーが持つ Color プロパティが自動でリスト表示されます。

### 5. カラーを設定する

モードに応じて、各ターゲットのカラー設定が表示されます。

#### Voice 連動モードのカラー設定

| 項目 | 説明 |
|---|---|
| **消灯** | 無音時（Voice = 0）のエミッションカラー。通常は黒（発光なし） |
| **点灯** | 発声時（Voice = 1）のエミッションカラー |

**強度倍率**（Voice 設定セクション）を上げると点灯色をさらに明るくできます（HDR 相当）。

#### Viseme カラーモードのカラー設定

「Viseme カラー詳細」を展開すると、口の形（0〜14）ごとのカラーを個別に設定できます。

| Viseme | 口の形 | 音の例 |
|---|---|---|
| 0: sil | 無音（口を閉じた状態） | — |
| 1: PP | 唇を閉じる | p, b, m |
| 2: FF | 上歯が下唇に触れる | f, v |
| 3: TH | 舌が歯の間に | th |
| 4: DD | 舌先が上歯茎に触れる | d, t, n |
| 5: kk | 舌の奥が上顎に触れる | k, g |
| 6: CH | 舌が上顎近くに上がる | ch, j, sh |
| 7: SS | 歯を閉じ気味にする | s, z |
| 8: nn | 鼻に抜ける | n, ng |
| 9: RR | 舌を丸める | r |
| 10: aa | 大きく開く | a |
| 11: E | 横に広げる | e |
| 12: ih | やや開く | i |
| 13: oh | 丸く開く | o |
| 14: ou | 唇を突き出す | u |

> **ヒント**
> `0: sil`（無音）は話していないときに常にアクティブです。光らせたくない場合は黒（発光なし）のままにしてください。

#### 切り替え時間（Viseme モード）

Viseme カラー詳細内の **切り替え時間 (秒)** を 0 より大きい値にすると、口の形が変わったときに前の色から新しい色へグラデーションのように変化します。

- スライダーで 0〜0.3 秒の範囲を操作できます
- 数値フィールドに直接入力すれば 0.3 秒を超える値も設定可能です
- 0.05〜0.1 秒あたりが自然に見えやすいです

### 6. セットアップ実行

**[セットアップ実行]** ボタンをクリックすると以下が自動生成されます：

- `Assets/LipSyncLight/Animations/` 以下に AnimationClip
- アバターの FX Controller に `LipSyncLight_Voice` / `LipSyncLight_Viseme` レイヤー

## カラーグループ機能

複数のターゲットで同じカラー設定を共有したい場合に使用します。

### グループの作成方法

**方法 A：「この設定でグループを作成」ボタン**

ターゲットのカラーを独自設定で作り込んだ後、ターゲットカード下部の **[この設定でグループを作成]** ボタンを押すと、現在の設定が新しいグループとして登録され、そのターゲットが自動でグループに切り替わります。

**方法 B：「+ グループを追加」ボタン**

カラーグループセクションの **[+ グループを追加]** で空のグループを作成し、手動でカラーを設定します。

### グループの使い方

各ターゲットの **消灯/点灯カラー** および **Viseme カラー** のドロップダウンから使用するグループを選択します。

- **独自設定**：そのターゲット専用のカラーを設定
- **グループ: ○○**：選択したグループのカラーを参照（変更不可でグレーアウト表示）

グループのカラーを変更すると、そのグループを参照しているすべてのターゲットに即座に反映されます。

> Voice と Viseme は別々にグループを設定できます。例えば「Voice は全ターゲット共通、Viseme はターゲットごとに個別」といった設定も可能です。

### コピー＆ペースト

各ターゲット・各グループにはコピー＆ペーストボタンがあります。セッション中のみ保持されるクリップボードを使って、カラー設定を別のターゲットやグループに素早く複製できます。

## 生成物の削除

**[生成物を削除]** ボタンで、生成した AnimationClip と FX Controller のレイヤーをまとめて削除できます。

## よくある質問

**Q. セットアップ実行後、VRChat にアップロードするだけで動きますか？**
A. はい。FX Controller に直接レイヤーを追加する方式のため、通常通りアップロードすれば動作します。

**Q. 既存の LipSync Light レイヤーがある状態で再実行するとどうなりますか？**
A. 上書き確認ダイアログが表示されます。「上書き」を選ぶと既存のレイヤーが置き換えられます。

**Q. lilToon / Poiyomi で使えますか？**
A. 使えます。Renderer を選択すると対応するエミッションプロパティ（`_EmissionColor` など）がドロップダウンに表示されます。

**Q. Voice 連動で声の強弱に応じて光り方が変わりますか？**
A. はい。`Voice` パラメーターは 0〜1 の float 値で、Blend Tree により消灯色と点灯色の間を連続的にブレンドします。強度倍率を上げると点灯時の明るさをさらに増幅できます。

**Q. Viseme カラーモードで声を出していないのに光ります。**
A. `0: sil`（無音）の Viseme カラーが黒以外に設定されているためです。常時光らせたくない場合は `0: sil` のカラーを黒にしてください。
